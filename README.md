# 🤖 Artificial Intelligence Internship - DecodeLabs 🚀

Welcome to my portfolio repository! This repository contains all the projects and practical milestones I have completed during my Artificial Intelligence Internship at DecodeLabs.

## 📁 Projects Overview

### Week 1: Rule-Based AI Chatbot
* **Concept:** Deterministic Control Flow, IPO Model, Hash Map (Dictionary) Lookup.
* **Goal:** Built a White-Box Guardrail chatbot capable of continuous interaction, sanitizing inputs, and O(1) constant time intent matching without relying on unstable if-elif ladders.

### Week 2: Data Classification (Supervised Learning)
* **Concept:** K-Nearest Neighbors (KNN), Feature Scaling, Data Splitting, Confusion Matrix.
* **Goal:** Trained a predictive machine learning model using `scikit-learn` on the Iris Benchmark dataset. Achieved high precision and recall (F1 Score) validated through a confusion matrix.

### Week 3: AI Recommendation Logic (Content-Based Filtering)
* **Concept:** TF-IDF Vectorization, Cosine Similarity, Feature Extraction, Vector Mapping.
* **Goal:** Built a Digital Matchmaker (Tech Stack Recommender) that maps raw user skills to optimal career paths by translating text into high-dimensional vector spaces and calculating angular alignment.

### Week 4: Machine Perception (Object Detection)
* **Concept:** Computer Vision, Transfer Learning, SSD Architecture, 4D Blob Construction, Confidence Thresholding.
* **Goal:** Engineered a robust Object Detection pipeline using OpenCV and a pre-trained MobileNet-SSD model. Translated raw visual matrices into decoded bounding boxes and class labels, successfully implementing a strict 80% confidence gatekeeper to prevent AI hallucinations.
